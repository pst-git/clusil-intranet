Access the filesystem (WebDAV)

The CLUSIL Member Intranet filesharing system is WebDAV based.

All CLUSIL Member Users have access to the WebDAV. Depending on the affiliation to a Working/Ad-Hoc Group, the same username and password gives access to the Group's documents via the following: https://intranet.clusil.lu/[wg-acronym] (acronym must be in lowercase!). To read on how to use the web interface to the file sharing platform, see below.

Alternatively, and for best user experience, use your standard "file manager" to "map a drive" to the Group's repository.

Find herebelow screenshots on how to set this up on Windows 7:

Dedicated WebDAV clients also exist for all known platforms. More details and resources about WebDAV at www.webdav.org

For existing CLUSIL Members, that want acceass to a new Group's documents, you can change/update your CLUSIL Member Profile.

If you're not a CLUSIL Member yet, use the registration application.
